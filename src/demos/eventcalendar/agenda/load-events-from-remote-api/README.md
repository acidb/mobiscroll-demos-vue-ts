## Demo description

The agenda can be populated by passing an array to the `data` option, that you can construct either inline or by getting it from a remote API. The important thing to remember is that events need to be [in a format that the agenda understands](https://demo.mobiscroll.com/vue/agenda/event-data-structure#).
